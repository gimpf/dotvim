dotvim
=======

some currently used vim configuration

Windows:

    git clone git://github.com/gimpf/dotvim "%USERPROFILE%\vimfiles"
    pushd "%USERPROFILE%\vimfiles"
    git submodule update --init --recursive

Linux:

    git clone git://github.com/gimpf/dotvim ~/.vim
    cd ~/.vim
    git submodule update --init --recursive
